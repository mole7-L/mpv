# mpv-subtitle-load

Automatically load subtitles from:

Subs/
    movie/
        01.ass
        02.ass

## Features

- Automatically detects subtitle directory
- Loads all subtitles
- Lightweight

## Installation

Copy subtitle_load.lua to:

~/.config/mpv/scripts/

## Directory Layout

Movies/
    movie.mp4

Subs/
    movie/
        01.ass
        02.ass
